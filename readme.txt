tips~

・重要变量首字母小写，其他首字母大写
例：imageCanny，imageGray，numberOfRectange，pixelGray
一般第一个词是名词，后面是对他的形容。
临时变量随意~~

・重要的变量可以用全局变量，例如imageCanny，这样子的话可以在任意的地方把图像显示出来，方便debug~
要是需要用到什么新的全局变量，直接添加到程序头上即可。

・重要函数、结构名、类型名首字母大写
例如：Init()，ImageProcess(), struct GraphPixel{}.....

・我在cpp文件里设置了3个pragma region，大家可以在自己的region里面添加代码，而且可以折叠别人的代码，看起来干净一些。当然也可以在别人的region里改，反正github会显示修改日志。各种算法以函数的方式呈现，main()函数中可以做各种测试。

・上传文件时要保证至少编译要通过，这样子别人可以直接运行~
如果暂时编译通不过，但是想上传保存的话，可以保存为其他文件名~

・先想到这些~要是想到什么大家都可以更新~
